# xonsh-dotfiles

My dotfiles for the Xonsh shell

## Install

To install pipx:

```shell
brew install pipx
```

To install xonsh:

```shell
pipx install 'xonsh[full]'
```

To install deps:

```shell
pipx inject xonsh pandas duckdb pyperclip
```

To install dotfiles:

```shell
git clone git@github.com:mattmc3/xonsh-dotfiles ~/.config/xonsh
```
